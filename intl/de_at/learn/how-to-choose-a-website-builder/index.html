So wählen Sie einen Website-Builder aus

Früher musste man zum Erstellen einer Website Code schreiben können oder jemanden kennen, der dazu in der Lage war. Dann kamen die Website-Builder: Dienstanbieter, die es jedem ermöglichen, mit einfachen Tools eine Website zu gestalten. Sie kümmern sich um den Code und bieten ansprechende Vorlagen, die Sie ganz einfach mit eigenen Texten, Designs, Bildern und anderen Komponenten anpassen können. Heute bauen sich Millionen Menschen ganz ohne Design- oder Programmiererfahrung mithilfe von Website-Buildern eine Präsenz im Internet auf.

Ist ein Website-Builder das Richtige für Sie?

Ein Website-Builder kann sowohl für große als auch für kleine Unternehmen genau die richtige Wahl sein. Beantworten Sie die folgenden Fragen, um herauszufinden, ob auch für Sie ein Website-Builder infrage kommt.

Wer gestaltet und pflegt die Website? Website-Builder sind eine gute Option, falls Ihre technischen Fähigkeiten nicht besonders ausgeprägt sind. So können Sie Änderungen direkt an der Website vornehmen, ohne einen Profi beauftragen zu müssen. Ein Website-Builder ist aber auch dann sinnvoll, wenn Sie eine Webdesignerin oder einen Webdesigner mit der Gestaltung Ihrer Website beauftragen und es dann später anderen Personen ermöglichen möchten, Inhalte zu aktualisieren.

Wie groß ist Ihr Budget? Website-Builder können eine kostengünstige Möglichkeit zum Erstellen einer Website sein. Viele dieser Website-Builder bieten ein kostenloses Basispaket, andere wiederum eine kostenlose Testversion. Danach fällt in der Regel eine monatliche Gebühr an, die von der Größe Ihrer Website abhängt und zusätzliche Dienste wie Webhosting abdecken kann.

Wofür wird Ihre Website verwendet? Website-Builder bieten oft Designs oder Vorlagen für viele verschiedene Branchen und Websitetypen, z. B. für Blogs oder Portfolios, Einzelhandel und E-Commerce, Veranstaltungen und Dienstleistungen. Wenn Sie wissen, welche Art von Website Sie möchten, können Sie den passenden Website-Builder auswählen.

Wie soll Ihre Website aussehen? Website-Builder bieten zwar viele Möglichkeiten, Websites an individuelle Wünsche anzupassen, es gelten aber dennoch einige Einschränkungen. Falls Sie Layouts oder Vorlagen verwenden möchten, die sonst niemand benutzt oder die sich vollständig personalisieren lassen, sollten Sie eine benutzerdefinierte Website gestalten. Möglicherweise müssen Sie dafür ein Team für Webentwicklung engagieren.

**Sind für Ihre Website spezielle Funktionen erforderlich? Website-Builder können mittlerweile verschiedenste Anforderungen erfüllen – von der Veröffentlichung von Newslettern bis hin zu E-Commerce. Die meisten optimieren Ihre Website auch automatisch für Mobilgeräte, was heutzutage unverzichtbar ist.

Checkliste für Ihren Website-Builder

Wenn Sie sich für die Nutzung eines Website-Builders interessieren, sollten Sie eine Liste mit Auswahlkriterien erstellen. Im Folgenden finden Sie einige Aspekte, die Sie dabei berücksichtigen sollten. Bringen Sie sie in eine Reihenfolge, die für Sie sinnvoll ist, und behalten Sie sie bei Ihren Überlegungen im Hinterkopf:

  • Nutzerfreundlichkeit
  • Beliebtheit
  • Preise
  • Flexibles Design
  • Designs/Vorlagen
  • Optimierung mobiler Websites
  • Support
  • Bloggen
  • E-Commerce
  • Newsletter erstellen
  • Mitgliedschaften erstellen
  • Mobile App
  • Webhosting

Welche Website-Builder kommen infrage?

Google Domains bietet eine Reihe vertrauenswürdiger Website-Builder sowohl mit kostenlosen als auch mit zahlungspflichtigen Optionen. Auf unserer Website-Builder-Seite erfahren Sie mehr über kostenlose Google-Dienste wie Blogger und Google Sites sowie kostenpflichtige Tools wie Squarespace und Wix, mit denen Sie sowohl ganz einfache Websites als auch umfangreiche Onlineshops erstellen können.

Einen ersten Eindruck von verschiedenen Website-Buildern können Sie gewinnen, indem Sie sich Websites ansehen, die damit erstellt wurden. Die meisten Dienste haben ein Portfolio mit Live-Websites, die auf ihren Vorlagen basieren.

Wenn Sie bereit sind, Ihre eigene Website zu gestalten, finden Sie den passenden Domainnamen auf domains.google.
